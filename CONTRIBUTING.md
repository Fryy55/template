# Contributing to this project
Thanks for taking the time to contribute! :purple_heart:

Since this is primarily a C++ project, it requires you to follow a number of standards. See [C++ Contribution Code Standards](https://github.com/Fryy55/Fryy55/blob/main/contribution/cpp.md) for more information

In case you can't contribute with code or enhancements, feel free to suggest a feature or report bugs/problems in the Issues tab

Starring or sharing this project with other is also a great way to contribute and it helps prioritize the development of it
